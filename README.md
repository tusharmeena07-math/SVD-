This project demonstrates the implementation of Singular Value Decomposition (SVD), a fundamental concept in linear algebra used in data compression, noise reduction, and machine learning.
The project includes mathematical explanation and code implementation.
Singular Value Decomposition (SVD) is a matrix factorization technique where any matrix A can be written as:

A = U Σ Vᵀ

where:
- U: orthogonal matrix (left singular vectors)
- Σ: diagonal matrix (singular values)
- Vᵀ: transpose of orthogonal matrix (right singular vectors)
A ∈ R^(m×n) ⇒ A = U Σ Vᵀ
- Language: Python 
- Libraries: NumPy / etc.
  The program decomposes a matrix into U, Σ, and Vᵀ and verifies the result by reconstructing the original matrix.
  This project helped me understand the geometric interpretation of SVD and its practical applications in data science.
